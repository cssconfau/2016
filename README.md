# CSSConf Australia 2016 Site
[2016.cssconf.com.au](http://2016.cssconf.com.au/)

## Install
```
npm i -g grunt-cli
npm i
gem install bundle
bundle install
```

## Build
```
grunt dev
```

And open [http://localhost:3333](http://localhost:3333).

## Release
```
grunt Release
```
